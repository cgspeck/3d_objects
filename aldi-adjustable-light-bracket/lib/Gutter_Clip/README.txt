# Summary

2 styles of gutter clips for Christmas Decorations

# Print Settings

Printer: Aurarum Wombot
Rafts: No
Supports: No
Resolution: 0.2mm, or 0.3mm
Infill: 25%

Notes: 
If you do 3 or 4 shells then it should be strong enough with 20 to 25% infill.  PLA will be fine, even in harsh Australian Summers but ABS might be a better option if you have dark colour gutters that get hot
